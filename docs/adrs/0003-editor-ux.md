# 0003 – UX dell’editor per traduttori

Status: Accepted

Context
- Flusso di traduzione intensivo di copia/incolla su campi lunghi.

Decisione
- Textarea comode (auto‑grow, line‑height), pulsanti Copia/Seleziona/Espandi.
- Scorciatoie: salva (Ctrl/Cmd+S), cicla “description” (Ctrl+Alt+D/Shift).
- Filtri: solo traducibili, solo modificati, filtro per path.
- Salvataggi rapidi (per‑campo) + globali, “Salva e Prec./Succ.”.

Conseguenze
- Maggior velocità e minor frizione per traduttori.
- JS leggero e locale, facile da mantenere.
